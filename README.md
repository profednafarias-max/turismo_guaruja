# 🏖️ Portal de Turismo - Guarujá (Pérola do Atlântico)

Bem-vindo ao repositório do **Portal de Turismo no Guarujá**. Este é um projeto de desenvolvimento web focado na criação de uma página informativa, moderna e totalmente semântica para turistas que desejam conhecer as melhores praias, gastronomia e pontos turísticos da cidade do Guarujá, SP.

---

## 🚀 Tecnologias Utilizadas

Para este projeto, foram utilizadas exclusivamente tecnologias nativas da web, focando em boas práticas de estruturação e estilização:

*   **HTML5:** Utilização de tags semânticas para garantir acessibilidade e otimização para motores de busca (SEO).
*   **CSS3:** Estilização baseada no conceito de *Box Model*, trabalhando com cores responsivas, tipografia moderna, sombras e bordas personalizadas.

---

## 🏗️ Estrutura Semântica do HTML

O projeto foi dividido utilizando as novas tags do HTML5 para que os navegadores e leitores de tela entendam perfeitamente a hierarquia do site:

*   `<header>`: Cabeçalho principal com o logotipo e o menu de navegação entre as seções.
*   `<main>`: Conteúdo principal e exclusivo da página.
*   `<section>`: Divisões temáticas da página (ex: Seção de Praias, Seção de Hotéis).
*   `<figure>` e `<figcaption>`: Para as fotos dos pontos turísticos e suas respectivas legendas informativas.
*   `<footer>`: Rodapé com direitos autorais, mapa do site e links para redes sociais.

---

## 🎨 Recursos de Estilização (CSS)

A identidade visual do site remete ao clima praiano, utilizando tons de azul e areia. Os seguintes conceitos foram aplicados:

*   **Tipografia:** Customização de tamanhos, pesos (`font-weight`) e estilos de fontes para facilitar a leitura.
*   **Plano de Fundo:** Uso de gradientes e imagens de fundo suaves para dar profundidade às seções.
*   **Bordas e Sombras:** Aplicação de `border-radius` nos cards de praias e `box-shadow` para criar um efeito de elevação (card design).
*   **CSS Inline e Interno:** Utilizado de forma estratégica para fixar conceitos de prioridade de renderização em elementos específicos.
